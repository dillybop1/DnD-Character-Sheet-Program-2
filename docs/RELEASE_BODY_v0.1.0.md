# D&D Character Sheet v0.1.0 Private Beta

This is the first private-beta desktop build for macOS and Windows.

## Included in this beta

- Roster-first single-window character workflow
- Dedicated saved-sheet page and editor route with live preview
- Offline local SQLite persistence
- JSON export/import for character transfer and backup
- Linked compendium and bounded homebrew support
- Print/PDF export

## Verified artifacts

- `DND Character Sheet-0.1.0-arm64.dmg`
- `DND Character Sheet Setup 0.1.0.exe`

## Important caveats

- This beta is intentionally unsigned on both macOS and Windows.
- macOS testers should expect a Gatekeeper warning on first launch.
- Windows testers should expect an unknown-publisher or reputation warning.
- Share this build only with trusted testers.

## Validation summary

- macOS packaged validation passed for install, launch, runtime/storage diagnostics, create/save/reopen, roster -> sheet -> edit routing, JSON export, PDF export, and reinstall-over-existing-data persistence.
- Windows packaged validation passed for create/save/reopen, JSON export/import, PDF export, and reinstall-over-existing-data persistence.
